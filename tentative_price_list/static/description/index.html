Tentative Price list

Tenative price list module for Odoo 11. This modules helps you set a target amount for a set of products in purchase order.

Step 1

Go to Tentative price list and create your price list. Choose the parent parent and set the target amount.

Step 2

Create your quotation and choose the price list.

Step 3

Choose the respective child products.

Step 4

Find the Remaining TPL amount and the purchased amount in Tentative price list as well.
