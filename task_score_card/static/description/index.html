Task Score Card

Overview:

Using this application, You can add score for each task if the assigned person does not complete it on time(On or Before the Deadline) the score will start reducing. You can easily assess the employee's performance since it also indicates the delay days.

Key Features:

  • Assessment Made easy
  • No Configurations needed
  • Be updated with the delays

Configuration

No Configurations Required.

How it Works

  • Create a Task, Give a Task Assigning Date and a Deadline for task score card to calculate Points.


  • As you can see in the below Image, The Points have been reduced to 80 since it has reached the deadline 2 days ago.


  • After 15 Days of not completing the task, It'll start displaying Negative Points.
