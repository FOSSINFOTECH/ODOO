COMPANY HOLIDAY MANAGEMENT

Overview:

This Application helps you add your list of company holidays for your users reference, Also reflects in the calendar module so you can schedule your meetings accordingly.

Key Features:

  • Reflects in the calendar Module
  • Add Filters
  • Default Email Notifications on holidays

Configuration

Make Sure your outgoing mail server is configured to send default notifications on holidays.

How it Works

Go to Employees --> Company Holidays to find the menu.

  • Holiday Creation

  • The Created Holiday reflecting in the calendar module.

  • Calendar View

  • Form View
