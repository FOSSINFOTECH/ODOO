Payslip Mass Mailing

The Main Idea:

This Module 'Payslip Mass Mailing' helps in sending bulk emails to employees at once, It e-mails the Payslip of the Employees in a PFD Format.

Key Features:

  • Bulk Mail
  • Emails Payslips In a Single click
  • Simple Configuration

Configuration

Make sure your outgoing mail server is configured.

How it Works

Once the payslips have been generated and confirmed, Select the payslips that you want to email and
Click on Action --> Payslip Mass Mailing as depicted in the below image.


A wizard pops-up, Click on 'Send Mail' to proceed with sending the payslips.
