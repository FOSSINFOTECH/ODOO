Purchase Revision

Overview:

This module stores up the previous quoted vaule on your RFQ, It can be used for future references.

It's very simple to use with no configuration and also provides you a pivot view for analysis.

Key Features:

  • Check your previous quoted price

  • Pivot view for report analysis

  • No configuration

  • Price Comparision.

Configuration

No configurations required.

How it Works

  • You'll find a button 'Create Revision' on the form.

  • Once you have quoted the prices for all the products

  • Click on 'Create Revision' it makes a copy of the line items from the products tab


  • To view the revision Go to 'View Revision'


  • For Pivot View : Go to Control --> Price History.
