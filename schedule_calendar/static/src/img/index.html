Schedule Calendar

Schedule Calendar module for Odoo 11. Using this module, You can create a scheduled event in the calendar and for any specific event.

Step 1

Install the module, Go to calendar menu and create an event.

Step 2

Or either create a task in the project menu.

Step 3

Now go to Schedule calendar, Find the Event and the Task that was created earlier which is now reflected in the schedule calendar.

Step 4

Go to schedule calendar, Click on 'View record'.

Step 5

Doing so, it will take you to the task you have created.
