Schedule Calendar

Overview:

You can Manage all your scheduled tasks and meetings in one place by installing this module.

Key Features:

  • Ease of use
  • Time Saver
  • Manage your tasks and meetings in one place

Configuration

No configurations required.

How it Works

  • If you schedule a meeting in the calendar module, it'll reflect the same in schedule calendar.


  • Reflection of the created meeting, both in form and calendar view.

  • Clicking on 'View Record' will take you to the original record.



  • The same action takes place if you create tasks.


Step 4

  • Reflection of the created task, both in form and calendar view.
