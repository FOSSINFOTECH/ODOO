Project Tags


Project Tags modules displays the status of the projects and tasks with tags, It also calculates the projected end date of the project and the respective task automatically. It captures the Actual Start Date and End Date.


Highlights


Status Indicator with Colors

The Status of each task is indicated with different colors.

Projected End Date

Projects Tags automatically predicts the end date of a project and task.


Actual Starting and Ending Date

The Acutal Starting and Ending Date is captured automatically, when clicked on start and end buttons.

Advantages

  • Project Tags helps you track your Project and Task status.
  • Project Tags displays color indicators for each stage.
  • Project Tags captures the actual starting and the ending date of both projects and tasks.
  • Project Tags predicts the end date of a project or a task based on the timesheets and the progress bar.

Benefits

  • It automatically captures the Actual Starting and Ending Date.
  • Projected End Date changes automatically both for projects and tasks.
  • Every stage has a unique color to display the status of projects and tasks
  • It helps you track the status of the projects and tasks without even opening the form, Since status is displayed on the Kanban View.

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise as well as Odoo Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.
