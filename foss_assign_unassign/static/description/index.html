Assign or unassign followers

Assign followers module for Odoo 11. Using this module, we can assign bulk number of followers to a particular record or N number of records. We can assign multiple followers to multiple of records using wizard.

Step 1

Settings > Technical > Action > Assign followers

Step 2

Create a Model for assigning Followers to records, Enter the Name and choose Model in the form. Save it and click “Add Action” button.

Step 3

It will be shown in the model you have chosen. Now you can assign a bulk number of followers to a particular record or N number of records.

Step 4

Now you can assign multiple followers to multiple of records using wizard.

Step 5

You can now see the followers you have added.
